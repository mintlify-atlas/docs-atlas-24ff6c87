The official website [analysis-tools.dev](https://analysis-tools.dev/) is powered by this repository and adds rankings, user comments, and video resources for each tool.

## How this catalog works

<Steps>
  <Step title="All tools are stored as YAML">
    Each tool has a YAML file in `data/tools/`. These files contain the tool's name, description, license, supported languages, and more.
  </Step>
  <Step title="A Rust renderer generates output">
    Running `make render` processes the YAML sources and generates both the README and a machine-readable JSON API.
  </Step>
  <Step title="A JSON API is available">
    The rendered output in `data/api/tools.json` can be used to build integrations, run queries with `jq`, or power your own tooling.
  </Step>
  <Step title="The community keeps it current">
    Anyone can submit a pull request to add or update tools. Deprecated or unmaintained tools are marked with a warning symbol.
  </Step>
</Steps>
